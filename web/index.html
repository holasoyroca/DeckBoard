DeckBoard v1.1.0

Panel de control y gestor web para optimizar y transferir archivos a tu consola.

¿Cómo subir juegos?

1. Ve a la pestaña de Gestor de ROMs.

2. Selecciona la consola objetivo de la lista lateral.

3. Arrastra tus archivos ROM a la zona de subida o haz clic para seleccionarlos desde tu explorador de archivos. Se subirán de forma inalámbrica a la carpeta de tu tarjeta SD/SSD.

¿Qué son los Shaders y Compatdata?

Shader Cache: Compilaciones gráficas precargadas por Steam para evitar tirones durante el juego.

Compatdata: Carpetas que emulan el sistema de Windows (prefijos Wine) y almacenan las partidas guardadas de cada juego.

DeckBoard analiza estas carpetas e identifica las que pertenecen a juegos que ya no tienes instalados en Steam (marcando los datos como Huérfanos) para que puedas borrarlos con seguridad.
